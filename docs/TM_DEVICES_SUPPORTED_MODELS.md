## How to Find Your Instrument

### Method 1: Search by Model Number
In the tm_devices browser, use the search box:
- Type: `"2460"` → Finds SMU2460
- Type: `"MSO6"` → Finds MSO6B Series
- Type: `"AFG"` → Shows all AFG models
- Type: `"DMM"` → Shows all DMM models

### Method 2: Browse by Category
Models are organized alphabetically but grouped by type:
1. Oscilloscopes (MSO, DPO, MDO) listed first
2. Function generators (AFG series)
3. Waveform generators (AWG series)
4. Source measure units (SMU series)
5. Digital multimeters (DMM series)
6. Other instruments

### Method 3: Know Your Series
If you know your instrument series:
- **6B Series?** → MSO6B, MSO5B, MSO4B
- **2600B Series?** → SMU2601B through SMU2636B
- **3000 Series AFG?** → AFG3K, AFG3KB, AFG3KC
- **7000 Series Scope?** → DPO7K, AWG7K

## Model Naming Convention

### Understanding Model IDs

**Format:** `[TYPE][SERIES][VARIANT]`

Examples:
- **MSO6B** = Mixed Signal Oscilloscope, 6 Series, B Variant
- **SMU2460** = Source Measure Unit, 2460 Model
- **AFG3K** = Arbitrary Function Generator, 3000 Series
- **DMM7510** = Digital Multimeter, 7.5 Digit, Model 10

### Series Suffixes
- **B** = Updated hardware revision (e.g., MSO6B)
- **K** = Series indicator (e.g., AFG3K = AFG3000 series)
- **Numbers** = Specific model variant

## tm_devices JSON Keys

Each model maps to a specific key in `tm_devices_full_tree.json`:

**Format:** `[lowercase_model]_commands.[MODEL]Commands`

Examples:
```
MSO6B    → mso6b_commands.MSO6BCommands
SMU2460  → smu2460_commands.SMU2460Commands
AFG3K    → afg3k_commands.AFG3KCommands
DMM6500  → dmm6500_commands.DMM6500Commands
```

## Frequently Asked Questions

### Q: My instrument isn't listed. What should I do?
**A:** Check if your instrument is part of a series that's supported. For example:
- MSO64 → Use MSO6B Series
- AFG3102 → Use AFG3K Series
- SMU2635A → Use SMU2635B (often compatible)

### Q: What's the difference between MSO6B and MSO6?
**A:** MSO6B is the newer "B Series" with updated hardware and firmware. Commands are often similar but the B series has additional features.

### Q: Can I use MSO6B commands on an MSO5B?
**A:** Some commands work across series, but many are model-specific. Always select your actual instrument model for best results.

### Q: Which AFG model should I select?
**A:** 
- AFG3022B/AFG3052C → Use AFG3K or AFG3KC
- AFG31000 series → Use AFG31K
- Check your instrument's front panel for the exact model

### Q: Are older instruments supported?
**A:** Yes! Legacy instruments like DPO2000, SMU2400 series, and older AFG models are included where tm_devices provides drivers.

## Adding New Instruments

If your instrument is not in the list but tm_devices supports it:

1. Check the tm_devices documentation for the exact driver class name
2. Add an entry to the `MODELS` array in `TmDevicesCommandBrowser.tsx`:

```typescript
{ 
  id: 'YOUR_MODEL', 
  label: 'Your Model Name', 
  rootKey: 'your_model_commands.YourModelCommands' 
}
```

3. Ensure the corresponding entry exists in `tm_devices_full_tree.json`
