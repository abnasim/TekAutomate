───────────────────────────────────────────────────────────────────
 TROUBLESHOOTING
───────────────────────────────────────────────────────────────────

Problem: "Node.js NOT Found" error

Solution:
  1. Open Command Prompt
  2. Type: node --version
  3. If "not recognized":
     - Install Node.js from https://nodejs.org/
     - During install, check "Add to PATH"
     - **Restart computer**
  4. Try SERVE.bat again

Problem: SERVE.bat still can't find npm/npx

Solution:
  The new SERVE.bat searches these locations:
  - System PATH
  - C:\Program Files\nodejs\
  - C:\Program Files (x86)\nodejs\
  - %APPDATA%\npm\
  
  If still not working:
  1. Find where Node.js is installed
  2. Edit SERVE.bat
  3. Add your custom path to the search locations

Problem: Port 3000 already in use

Solution:
  - Edit SERVE.bat
  - Change "3000" to "3001" or another port
  - Line near bottom: "npx serve -s build -l 3000"
